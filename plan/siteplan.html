Reason for project: This project is also a valuable learning opportunity. By building a fully functional website from scratch, I will apply the theoretical knowledge I have gained throughout the course in a practical setting. Developing the skills to fetch data from external APIs, utilize JavaScript for dynamic content, and implement localStorage for user preferences will deepen my understanding of web development

Site Purpose

The purpose of the website is to offer users an engaging platform to discover, search, and interact with a database of movies and TV shows. The site will provide users with search capabilities, personalized features (such as favorites), and dynamic content fetched from external APIs for trending and popular shows.

Scenarios

Scenario 1: "How can I find the latest trending movies and TV shows?"

Scenario 2: "How can I save and manage my favorite movies and shows on the website?"
